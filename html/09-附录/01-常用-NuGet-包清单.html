常用 NuGet 包清单 ​

📦 MediatR 生态系统中的核心包和扩展

🎯 核心包 ​

MediatR ​

包名: MediatR
最新版本: 12.x
用途: MediatR 核心库

bash
dotnet add package MediatR
1

包含内容:

  • IMediator 接口
  • IRequest<T> / IRequestHandler<T, R>
  • INotification / INotificationHandler<T>
  • IPipelineBehavior<T, R>
  • 默认的 Mediator 实现

MediatR.Extensions.Microsoft.DependencyInjection ​

包名: MediatR.Extensions.Microsoft.DependencyInjection
最新版本: 11.x
用途: ASP.NET Core 依赖注入集成

bash
dotnet add package MediatR.Extensions.Microsoft.DependencyInjection
1

包含内容:

  • AddMediatR() 扩展方法
  • 自动扫描和注册 Handler
  • 自动注册管道行为

注意: MediatR 12+ 已将此功能合并到主包中，可以不再需要此包。

用途: 强大的验证框架

🗺️ 对象映射 ​

AutoMapper ​

包名: AutoMapper.Extensions.Microsoft.DependencyInjection
最新版本: 12.x
用途: 对象到对象的映射

bash
dotnet add package AutoMapper.Extensions.Microsoft.DependencyInjection
1

使用场景:

  • 领域模型 → DTO 转换
  • 在 Handler 中简化数据转换

示例:

csharp
builder.Services.AddAutoMapper(typeof(Program).Assembly);

public class GetOrderHandler : IRequestHandler<GetOrderQuery, OrderDto>
{
    private readonly IMapper _mapper;
    
    public async Task<OrderDto> Handle(GetOrderQuery request, CancellationToken ct)
    {
        var order = await _repo.GetByIdAsync(request.OrderId);
        return _mapper.Map<OrderDto>(order);
    }
}
